Music Transposer

Transpose notes and chords between different keys

How to Use:

  • Select the original key and the target key
  • Enter notes or chords in the input box (separated by commas or spaces)
  • Click "Transpose" to see the result
  • Examples: C, F, G7, Am, Dm7, E7#9

Example: If you input "C, F, G" and transpose from C to G, the result will be "G, C, D"

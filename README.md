# Key Transposer

Key Transposer is a simple, browser-based tool that allows musicians to transpose chords and notes from one key to another instantly.  
It supports sharps, flats, major/minor chords, and common chord extensions.

## Features
- Transpose chords or notes between any two keys
- Supports sharps (#), flats (b), and chord extensions (7, maj7, m7, etc.)
- Copy results directly to clipboard
- Mobile-friendly and responsive design
- No installation required â€” runs in any modern browser

## How to Use
1. Open Key Transposer in your browser.
2. Select the original key and the target key.
3. Enter chords or notes in the input box (comma or space separated).
4. Click "Transpose" to see the result.
5. Optionally, click "Copy Result" to copy the transposed chords.

## Deployment
This tool is hosted on [Netlify](https://key-transposer.netlify.app).
